# README

## Installing

```sh
curl -sO https://raw.githubusercontent.com/peterotool/.dotfiles/main/dotfiles
```

## Main Scripts

* scripts/config.sh: general settings and dotfiles
* scripts/osx.sh: MacOS settings

## Testing Stow
